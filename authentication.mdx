---
title: "Authentication"
description: "Overview on how to authenticate your requests"
---

Every request will be authenticated before processing for the response. A request should a containe a bearer token in the headers for authentication, like the example given below:

```bash
'Authorization': 'Bearer <team_token>'
```

## API Tokens

For generating the API token, first user has signup to our mobile application. Once the account is created successfully, API token will be shared by the support team.
