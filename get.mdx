---
title: "Get Past Data"
description: "This endpoint provides logs data for a certain period.
This endpoint only supports 6 pin smifi devices."
---

### Query parameters

<ParamField body="deviceid" type="string">
  This is id of the device for which data will be fetched.
</ParamField>

<ParamField body="startTime" type="number">
  The period of required data will start from time provided in startTime. It
  should be represented in milliseconds. And the startTime shall not be earlier
  that 30 days before the request.
</ParamField>

<ParamField body="endTime" type="number">
  This represents the end of the period for which data is requested. It should
  be represented in milliseconds.
</ParamField>

### Response

<ResponseField name="id" type="string">
  This is the id of the device for which data is provided.
</ResponseField>

<ResponseField name="timestamp" type="number">
  This is the timestamp when the data was generated. (timestamp / 10000)
</ResponseField>

<ResponseField name="volt" type="number">
  Indicates the voltage of the supply to the device.
</ResponseField>

<ResponseField name="curr1" type="double">
  Indicates the value of the current on the pin 1 of the device.
</ResponseField>

<ResponseField name="curr2" type="double">
  Indicates the value of the current on the pin 2 of the device.
</ResponseField>

<ResponseField name="curr3" type="double">
  Indicates the value of the current on the pin 3 of the device.
</ResponseField>

<ResponseField name="curr4" type="double">
  Indicates the value of the current on the pin 4 of the device.
</ResponseField>

<ResponseField name="curr5" type="double">
  Indicates the value of the current on the pin 5 of the device.
</ResponseField>

<ResponseField name="curr6" type="double">
  Indicates the value of the current on the pin 6 of the device.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://epviapi.in/devicelogs?deviceId=epvitest&startTime=968437569&endTime=947382947' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--query-parameter '{
  "deviceId": "kbhfkdshbfslhf",
  "startTime": 9684375697483,
  "endTime": 9473829471425
}'
```
