---
title: "Live Data"
description: "This endpoint makes incoming data live to client."
---

### Request

<ResponseField>
  Receiving request via socket.io https://socket.io/docs/v4/
</ResponseField>

<ResponseField name="deviceids" type="array">
  deviceids is an array of strings in the query field of headers.
</ResponseField>

<ParamField body="token" type="string">
  Token is a string in the Auth of header.
</ParamField>

<RequestExample>
```bash Example Request
curl --location --request  'https://epviapi.in/livedata' \
--header 'Auth:<token>' \
--header 'Query: deviceids' \
```
</RequestExample>

### Response

<ParamField body="voltage" type="number">
  This field shows value of voltage.
</ParamField>
<ParamField body="deviceid" type="string">
  This field shows the deviceid.
</ParamField>

<ParamField body="current1" type="number">
  This field shows current in pin 1.
</ParamField>

<ParamField body="current2" type="number">
  This field shows current in pin 2.
</ParamField>

<ParamField body="current3" type="number">
  This field shows current in pin 3.
</ParamField>

<ParamField body="current4" type="number">
  This field shows current in pin 4.
</ParamField>

<ParamField body="current5" type="number">
  This field shows current in pin 5.
</ParamField>

<ParamField body="current6" type="number">
  This field shows current in pin 6.
</ParamField>
